Table

Lightweight, data-driven table

Render tabular data with a clean, minimal API.

Install & Import

Dependencies: Bootstrap Icons, Bootstrap Table (CSS/JS), TableExport, FileSaver.

Quick Start
const tbl = new Table({ 
  name: 'Users',
  options: {
    search: true,
    pagination: true,
    pageSize: 10
  }
});
tbl.renderInto('container-id');
tbl.updateData([
  { id: 1, name: 'Alice', role: 'Admin' },
  { id: 2, name: 'Bob', role: 'User' }
]);
Options
  • search: Enable search input (default: true).
  • export: Enable data export dropdown (default: true).
  • pagination: Enable pagination controls (default: true).
  • pageSize: Rows per page (default: 25).
  • height: CSS height string (default: '460px').
  • fields: Optional array of column keys to display.
  • link: Config object { key, fields, listen } to make columns clickable.
API
  • renderInto(id): Renders the table structure.
  • updateData(rows | row): Loads data into the table. Auto-detects columns if not set.
  • clear(): Removes all data from the table.
  • handleSearch(query): Programmatically filters the table.
  • activatePage(page): Switches to the specified page number.
Events

Events are configured via the link option.

// Example: Emit 'user:click' when 'id' column is clicked
const tbl = new Table({
  name: 'Users',
  options: {
    link: {
      key: 'id',              // Column to make clickable
      fields: ['id', 'name'], // Data to pass in event
      listen: 'user:click'    // Event name
    }
  }
});

tbl.on('user:click', (data) => console.log('Clicked:', data));
Features
  • Auto-column detection: Columns inferred from first data row if fields not specified.
  • Search filtering: Live search with debouncing across all columns.
  • Export: Export visible (filtered) data to JSON or CSV (requires jQuery & tableExport libs).
  • Nested objects: Automatically displays JSON objects with expandable view icon.
  • Row numbering: First column shows dynamic row numbers that update with filtering.
  • Pagination: Navigate large datasets with page controls and info display.
