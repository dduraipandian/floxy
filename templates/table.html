Quick Start
const tbl = new Table({ 
  name: 'Users',
  options: {
    search: true,
    export: true,
    pagination: true,
    pageSize: 25
  }
});
tbl.renderInto('table-demo');
tbl.updateData([
  { id: 1, name: 'Alice', role: 'Admin' },
  { id: 2, name: 'Bob', role: 'User' }
]);
Options
  • search: Enable search input (default: true).
  • export: Enable export dropdown (JSON/CSV) (default: true).
  • pagination: Enable pagination (default: true).
  • pageSize: Rows per page (default: 25).
  • fields: Array of field names to display; auto-detects from data if empty.
  • height: Table height in pixels (default: '460').
  • link: Link configuration object: { key, fields, listen } for clickable cells.
  • classNames: Additional CSS classes for the table.
API
  • renderInto(id): Renders table into container.
  • updateData(rows): Updates table with array of row objects. Auto-detects columns.
  • clear(): Clears table header and body.
  • handleSearch(query): Programmatically filter rows by search term.
  • activatePage(pageNumber): Jump to specific page.
  • refreshPagination(): Recalculate and redraw pagination.
Link Configuration

Configure clickable cells that emit events with row data:

const tbl = new Table({
  name: 'Users',
  options: {
    link: {
      key: 'id',              // Column to make clickable
      fields: ['id', 'name'], // Fields to pass in event
      listen: 'user:click'    // Event name with placeholders
    }
  }
});
tbl.renderInto('container');
// Emits event with row data when id cell is clicked
Features
  • Auto-column detection: Columns inferred from first data row if fields not specified.
  • Search filtering: Live search with debouncing across all columns.
  • Export: Export visible (filtered) data to JSON or CSV (requires jQuery & tableExport libs).
  • Nested objects: Automatically displays JSON objects with expandable view icon.
  • Row numbering: First column shows dynamic row numbers that update with filtering.
  • Pagination: Navigate large datasets with page controls and info display.
