Dropdown

Minimal, searchable, animated menu

A compact dropdown with optional search, item animations, and event-driven selection.

Install & Import

Use as an ES module. Bootstrap CSS/JS already included in this workspace.

<script type="module">
import Dropdown from './components/dropdown.js';
</script>
Quick Start
const dd = new Dropdown({
  name: 'Select Item',
  options: {
    maxHeight: '15em',
    search: true
  }
});

dd.renderInto('dropdown-demo');
dd.setDropdownItems([
  { name: 'Alpha', value: 'A' },
  { name: 'Bravo', value: 'B' },
  { name: 'Charlie', value: 'C' }
]);

dd.on('item:click', (item) => console.log('Selected', item));
Options
  • maxHeight: Max height for list scrolling (default 15em).
  • search: Enables search input and filtering.
API
  • renderInto(id): Renders dropdown into container.
  • setDropdownItems(items): Sets menu items; expects array of { name, value }.
  • on('item:click', cb): Subscribe to selection events.
  • on('init', cb): Fired once on first dropdown open.
Events
  • init: Emitted on first open of the dropdown (useful to lazy-load items).
  • item:click: Emitted when a user selects an item; receives the item object.
const dd = new Dropdown({ name: 'Choose', options: { search: true } });
dd.renderInto('container');
dd.on('init', () => {
  // fetch or set items lazily
  dd.setDropdownItems([{ name: 'One', value: 1 }]);
});
dd.on('item:click', (item) => console.log('Selected', item));
