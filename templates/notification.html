Notification

Notification + error modal notifications (Bootstrap based)
Install & Import

Use as an ES module. The library exports a default `notification` instance. 

<script type="module">
import notification from './components/notification.js';
</script>
Quick Start
import notification from './components/notification.js';
// Basic notifications
notification.info('Informational message');
notification.success('Saved successfully');
notification.warning('Caution: check this');
// Error with modal details
notification.error('An error occurred', 'Error Title', { code: 500, message: 'Server error' });
