Split

Resizable panes layout

Two- or multi-pane layout with a draggable divider.

Install & Import

Use as an ES module.

<script type="module">
import SplitPane from './components/split.js';
</script>
Quick Start
// Create container and pane elements first
const topDiv = document.createElement('div');
topDiv.innerHTML = "Top Content";
const bottomDiv = document.createElement('div');
bottomDiv.innerHTML = "Bottom Content";

const split = new SplitPane({
  containerID: 'split-container',
  top: topDiv,
  bottom: bottomDiv,
  options: {
    min_size: 100
  }
});
split.render();
Options
  • containerID: ID of the host element.
  • top: HTMLElement for the top pane.
  • bottom: HTMLElement for the bottom pane.
  • options.min_size: Min size in px for panes (default: 60).
  • options.topHeight: Initial top pane height (default: '49.9%').
API
  • render(): Renders the split layout into the container.
  • init(): Re-initializes event listeners (internal).
Events

This component does not currently emit events.
