Split

Resizable panes layout

Two- or multi-pane layout with a draggable divider.

Install & Import
<script type="module">
import Split from './components/split.js';
</script>
Quick Start
const split = new Split({ name: 'Main Split' });
split.renderInto('split-demo');
split.setPanes([
  { id: 'left', content: 'Left pane' },
  { id: 'right', content: 'Right pane' }
]);
API
  • renderInto(id): Renders split layout into container.
  • setPanes(panes): Array of panes { id, content }.
  • on('resize', cb): Subscribe to pane resize events.
