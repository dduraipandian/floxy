Tab

Simple, configurable tabs

Create tabbed views with a minimal API.

Install & Import

Use as an ES module.

<script type="module">
import Tab from './components/tab.js';
</script>
Quick Start
const tab = new Tab({ 
  name: 'my-tabs',
  options: {
    activeTab: 'home' 
  }
});
tab.renderInto('container-id');
tab.addTab({ id: 'home', title: 'Home', content: 'Home Content' });
tab.addTab({ id: 'profile', title: 'Profile', content: 'Profile Content' });
Options
  • tabs: Initial array of tab objects (optional).
  • activeTab: ID of the tab to show initially.
API
  • renderInto(id): Renders the tab layout into the target container.
  • addTab(tab): Adds a new tab. Expects { id, title, content }. Content can be a string, HTMLElement, or Component instance.
Events

This component currently relies on native Bootstrap tab events on the DOM elements.
